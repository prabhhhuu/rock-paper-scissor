Rock Paper Scissors Game
A simple and interactive Rock-Paper-Scissors game built with HTML, CSS, and JavaScript. Users can play against the computer by clicking on images representing each move. The game tracks wins, losses, and ties using local storage, and includes a reset button to clear the score.

Features:

Clickable image buttons for Rock, Paper, and Scissors.

Real-time result display (win, lose, or tie).

Score tracking with local storage.

Clean and responsive UI.
